bulma-cool-checkboxes

Based on hunzaboy/Cool-Checkboxes-for-Bulma.io. Uses Font Awesome 5.

View on Github

npm install @cityssm/bulma-cool-checkboxes

Supports bulma modifier classes.
e.g. is-primary, is-success, is-warning, is-info, is-danger, is-dark, is-black

Cool Checkboxes

Inline Checkboxes

Markup

<div class="field">
    <p class="control">
        <div class="b-checkbox is-primary">
            <input id="checkbox" checked type="checkbox" />
            <label for="checkbox">
                Is Primary
            </label>
        </div>
    </p>
</div>
